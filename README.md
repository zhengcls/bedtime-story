# 宝贝晚安 · 每日睡前故事

一个精美的儿童睡前故事播放器，每天自动更换一个温暖的睡前故事，配有女声朗读、星空动画和逐字显示效果。

## 功能特性

- **7个内置故事**（共42个完整章节），每天自动轮换
- **女声朗读** - 基于 Web Speech API，自动选择中文女声
- **星空背景** - Canvas 实时绘制星星闪烁 + 流星划过
- **月亮与萤火虫** - CSS 动画营造温馨氛围
- **逐字打字机效果** - 文字配合语音同步显示
- **多章节导航** - 自由切换章节，支持自动连播
- **朗读速度调节** - 慢 / 正常 / 快 三档
- **响应式设计** - 手机、平板、电脑均可完美显示
- **零依赖** - 纯 HTML+CSS+JS，单文件部署

## 在线访问

通过 GitHub Pages 部署后访问：`https://你的用户名.github.io/bedtime-story/`

## 本地预览

直接在浏览器打开 `index.html` 即可。

## 部署到 GitHub Pages

1. 将此仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后等待部署完成

## 技术说明

- 语音朗读使用浏览器原生 Web Speech API，无需第三方服务
- 首次使用需点击"开始听故事"按钮（浏览器要求用户交互后才能播放音频）
- Chrome/Edge 效果最佳，推荐使用现代浏览器
